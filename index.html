Guía de Revisión de JavaScript

Proyecto ToDo con localStorage

1. Fundamentos

Antes de empezar el proyecto, revisemos:

  • Funciones en JavaScript (MDN)
  • var, let y const (freeCodeCamp)
Ejercicio: Escribe una función que reciba el nombre de un alumno y devuelva un saludo. Prueba declararla con function y con const usando arrow functions.

2. Arreglos y Objetos

Las tareas de tu ToDo estarán en un arreglo. Revisa:

  • Array – MDN
  • Más de de Arrays (freeCodeCamp)
  • Ejemplos de Arrays (freeCodeCamp)
  • Objetos – MDN
Ejercicio: Declara un arreglo de objetos que representen tareas:
[{id:1, texto:"Estudiar JS", completada:false}, ...].
Agrega una nueva tarea al arreglo y muéstrala en consola.

3. DOM y Eventos

Necesitarás interactuar con botones e inputs:

  • Introducción al DOM – MDN
  • addEventListener – MDN
  • Eventos en JavaScript – freeCodeCamp
Ejercicio: Crea un botón en HTML. Con JavaScript agrega un evento click que muestre en consola: "¡Botón presionado!".

4. Persistencia con localStorage

Para guardar las tareas incluso si cierras el navegador:

  • localStorage – MDN
  • Ejemplos con localStorage – freeCodeCamp
Ejercicio: Guarda en localStorage tu nombre bajo la clave "alumno". Después recupéralo y muéstralo en un alert().

5. Mejoras y Refactorización

Cuando tu ToDo funcione, mejora el código:

  • filter – MDN
  • map – MDN
  • Template literals – freeCodeCamp
Ejercicio: Implementa un botón "Eliminar completadas". Usa filter() para quedarte solo con las tareas no completadas.
